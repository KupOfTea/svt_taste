# 세븐틴 취향표 - Flutter Web
#### [사이트 링크](https://svt-style.web.app/)
---
Flutter Web으로 제작했던 [세븐틴 취향표](https://svt-style.web.app/)입니다.
---
Flutter Web을 통해 모바일에서 이미지 저장이 구현되지 않는 이슈로 인해
이 프로젝트 이후 취향표 생성기 시리즈는 **Next.js, TypeScript, Tailwind CSS를 이용하여** 제작되고 있습니다.
